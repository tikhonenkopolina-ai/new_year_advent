# Telegram Advent Bot (Sequential)

- Кнопка: «Что сегодня?»
- Дни открываются по порядку с Дня 1 (прогресс хранится в state.db)
- На день: 1 текст, затем медиа подряд без подписей
- Render Free + Webhooks
- Команда для теста: /reset (сбрасывает прогресс в этом чате)


## Важно про дни
День определяется по календарю от `ADVENT_START_DATE` (по умолчанию 2025-12-26). Это сделано, чтобы на Render Free прогресс не сбрасывался при перезапусках.
Для теста можно временно поставить `ADVENT_START_DATE` равным сегодняшней дате.
